# 🛒 ShopSystem

Full E-Commerce System built with Laravel.

## 🚀 Features

- User Authentication (Login / Register)
- Admin Dashboard
- Product Management (Add / Update / Delete)
- Shopping Cart System
- Order Management
- Delivery System

## 🛠️ Technologies Used

- Laravel
- MySQL
- Blade
- PHP

## 👤 Roles

- Admin
- Customer
- Delivery
